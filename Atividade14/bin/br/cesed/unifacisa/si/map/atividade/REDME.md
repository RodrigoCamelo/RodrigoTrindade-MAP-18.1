Começando Go é bem mais simples e intuitivo que Java, enquanto Go não precisa de Gets e Sets, Java necessita desse métodos de manipulação. Outra coisa é que Go precisa importa o fmt, Java não precisa desse importe. A classe Filho recebe as variáveis nome e idade da classe Pai, mas em Java é preciso implementar elas na classe Filho. Em Go as Classes Pai e Filho não herda da interface Familia, mas em Java eles precisam herda.
